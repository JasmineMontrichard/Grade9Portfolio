Australia

What did the assignment require me to do?

The assignment required me to make a website about Australia and some of its states. I had to add sections of the different states with an image floating next to it.

What did I learn?

I learned that:

  • You can make a header using <header> instead of the <div> tag. I experimented with the <header> tag; using an image for the header looked a lot better than using any text tags (<p>, <h1>, etc.) as <header> had no effect on it.
  • You can use the border-radius property and edit an element's specific corner(s).

    Example

    • border-radius: 2px; (Defines the outer border edge by 2px)
    • border-radius: 5px 5px 0 0; (Defines the top-left and top-right outer border edges by 5px)

The Universe

What did the assignment require me to do?

The assignment required me to make a webpage explaining our galaxy, the solar system and the planets in it. We had include lists, subscript and superscript, different level-headers, and emphasis.

What did I learn?

I learned that:

  • You can change different list types and the order of them (starting from number 2, changing list type to alphabets, etc.).
  • You can use the <sub> and <sup> tags instead of copying and pasting subscript/superscript numbers.
  • You can use character entities instead of copying and pasting them from the internet into my files.

Unit 1 - Tag Examples

What did the assignment require me to do?

The assignment required me to make a webpage displaying the tags we've learned, while also copying exactly what the final result should look like.

What did I learn?

I learned that:

  • You can use the <pre> tag for saving exactly what was put inbetween them, including spaces.

  • There is a spacing difference when including <br> in the <p> tags and outside of it.

Sailing Adventures

What did the assignment require me to do?

The assignment required me to make a website about sailing, with information about sailing gear and tourist information. The website needed to have a list for sailing gear, a section for tourist information, and a footer with important links.

What did I learn?

I learned that:

  • You can change the style of the <nav> tag by changing the list-style-type property. This could also be used for other list tags (<ul>, <ol>, etc.).
  • You can use different tags to define sections other than <div> such as: <section> and <article>.
  • You can use the id attribute to name different elements on your webpage (<p>, <div>, <h1>, etc.). This can be useful for navigating to different parts of your webpage.
