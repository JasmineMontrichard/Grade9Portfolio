Key Learnings

Observations about my own learning:

When I am Stuck:

  • I can ask someone for help.
  • Reference my notes.
  • Search the internet for any solutions.

Learning from Specific Mistakes:

  • For my Universe assignment, I put the <br> tag after the <p> tag and was confused on why spacing was a lot greater than what was shown in the assignment's output. I put the <br> tag before the <p> ending tag, and I learned that there was a spacing difference.
  • For my Unit 2 test, there was a question asking about bullet points and its appearance for list items. After my test was graded, I learned that the only way you can change how bullet points look is through CSS.
  • For one of my learning tasks in Unit 4, we had to add the float property to the image. But I misunderstood the instruction and added the float property to the header above the image, and I became confused. I learned that the image will go beside the header and not below it.

Observations About the Importance of Practice:

  • It will be easier for me to remember tags, including their usage and syntax when writing my code.
  • It will help me practice good coding habits, such as: using external css instead of internal, using classes and not the style attribute.
  • It will help me find better solutions when writing code.

Formatting Issues:

  • I made a mistake where I was trying to style a navigation bar without affecting the other lists. I used a <ul> tag for the navigation bar and was trying to find a solution for having the style only apply to the navigation bar. I looked through my notes and remembered that you can make styles for restricted classes.
  • I used the W3 HTML & CSS Validators to check my code for any issues and suggestions that I can make when I am writing my code.
  • I had an issue where I was trying to add spacing to the credits in my footer. I was going to have © Copyright 2021  Jasmine Montrichard as the content for the footer and want to have spacing between © Copyright 2021 and my full name. I tried using the <pre> tag and I did not like the font it gave the text. I remembered that there are character entities for different spaces, so I looked up the unicode spacing size comparison chart. Then, I typed the character entity and it looked the way I intended it to be.
